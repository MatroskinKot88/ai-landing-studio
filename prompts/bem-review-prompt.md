### Промпт для генерации блока отзывов ###

Ты — Senior Frontend-разработчик с опытом создания B2B-лендингов для HoReCa-индустрии.  
Контекст: Я разрабатываю лендинг для компании, производящей барные станции и рабочие столы для ресторанов. Целевая аудитория — владельцы баров, шеф-бармены, управляющие. Мне нужен блок с реальными, правдоподобными отзывами, усиливающими доверие.  
Задача: Создай HTML-разметку блока с 3 отзывами клиентов — владельца бара, шеф-бармена и управляющего рестораном. Каждый отзыв должен содержать: имя, должность, фото (заглушка через data-src), текст отзыва (2–3 предложения, конкретика: срок службы, удобство, поддержка).  
Формат вывода: Только валидный HTML без CSS. Используй строгую БЭМ-нотацию. Корневой блок — reviews.  
Ограничения: Не используй изображения напрямую (только data-src="..."), не подключай JavaScript, не применяй фреймворки. Тексты отзывов должны звучать как от реальных людей, а не шаблонно.

### Почему работает ###

Роль задаёт уровень качества (senior + B2B + HoReCa).
Контекст объясняет зачем нужны отзывы → ИИ делает их конкретными.
Формат и ограничения гарантируют чистый, встраиваемый код.
БЭМ — явно указан корневой блок reviews, значит, структура будет предсказуемой.
